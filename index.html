Research

My current research focuses on Adaptive Memory Systems for AI, that create architectures that allow intelligent agents to store, prioritize, and retrieve information based on relevance, context, and temporal cues.

The main objectives are to:

  • Develop scoring algorithms that merge recency decay, novelty detection, entropy, and user feedback into a single metric.
  • Implement a memory storage: short-term buffers for immediate context, long-term vector databases for semantic recall, and archival layers for compressed, rarely accessed knowledge.
  • Integrate AES-GCM so that stored data remains secure without sacrificing the ability to perform intelligent retrieval.

This work draws on research from Neural Turing Machines, Differentiable Neural Computers, continual learning strategies like Elastic Weight Consolidation, and modern memory-augmented large language models.

Projects

CamSync File Management System

CamSync is my flagship project. This is a file management platform that combines AES-256-GCM encryption with AI based document summarization. I built CamSync using Flask, Nginx, and PyTorch based transformers. Every file is encrypted before storage and decrypted only for authorized users. The AI summarizer gives user(s) a concise and context-aware summaries for multiple file formats without needing to open them. I'm currently looking at how my research can be integrated into CamSync, to enable it to learn from user interaction patterns to improve search accuracy and prioritize relevant files.
