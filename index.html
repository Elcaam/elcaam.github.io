Research

My current research focuses on Drift Detection in Retrieval-Augmented Generation (RAG) Systems. The project investigates how AI knowledge bases evolve over time and how these changes affect retrieval and generation quality. The goal is to design a lightweight framework that continuously monitors RAG pipelines to detect "knowledge drift," ensuring outputs remain accurate and reliable even as underlying data changes.

Key objectives include:

  • Building an open-source RAG pipeline using public datasets and pre-trained models to establish a baseline for retrieval and generation.
  • Developing drift metrics at both the retrieval and generation levels, including embedding similarity, Recall@K, and semantic consistency between outputs.
  • Implementing scripts to track drift over time as documents are added, modified, or removed from the knowledge base.
  • Visualizing drift patterns and analyzing trends to highlight potential degradation in AI performance.
  • Optionally extending the work with formal modeling and statistical analysis to understand drift dynamics more rigorously.

This research draws on concepts from AI monitoring, knowledge base management, information retrieval, and statistical analysis, combining applied experimentation with opportunities for theoretical exploration. It represents a step toward building AI systems that remain robust, reliable, and contextually aware as they interact with evolving data.

Projects

CamSync File Management System

CamSync is my flagship project. This is a file management platform that combines AES-GCM with AI-based document summarization. I built CamSync using Flask, Nginx, and PyTorch-based transformers. Every file is encrypted before storage and decrypted only for authorized users. The AI summarizer gives user(s) concise and context-aware summaries for multiple file formats without needing to open them.
